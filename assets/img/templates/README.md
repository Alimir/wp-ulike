# Template Images Directory

This directory should contain template preview images for Pro templates.

## Required Images

The following SVG images should be copied from the Pro version:
- `default.svg` - Simple Up/Down Vote template
- `bookHeart.svg` - Book Heart template
- `checkMark.svg` - Check Mark template
- `voters.svg` - Voter Thumb template
- `checkVote.svg` - Check Vote template
- `brokenHeart.svg` - Broken Heart template
- `posNeg.svg` - Positive/Negative Circles template
- `feedback.svg` - FeedBack template
- `ratingFace.svg` - Rating Face template
- `ratingBoy.svg` - Rating Boy template
- `ratingGirl.svg` - Rating Girl template
- `stackVotings.svg` - Up/Down Votes template
- `starThumb.svg` - Star Thumb template
- `arrowVotings.svg` - Arrow Votings template
- `minimalVotings.svg` - Minimal Votings template
- `badgeThumb.svg` - Badge Thumb template
- `faveStar.svg` - Fave Star template
- `pin.svg` - Pin Button template
- `clapping.svg` - Clapping Button template
- `smileySwitch.svg` - Smiley Switch button template

## Source Location

Copy these images from:
`/wp-content/plugins/wp-ulike-pro/assets/img/templates/`

## Note

If images are not present, the system will attempt to use Pro version images if Pro is active, or display placeholders in the admin interface.
